# 🏏 IPL Player Performance & Auction Insights (2008–2016)

## 🚀 Overview
This project analyzes IPL player performance data (2008–2016) to derive insights that can support **team selection and auction strategy decisions**.

Using Tableau, the analysis focuses on understanding:
- Batting performance trends
- Player impact across roles
- Bowling behavior in different match situations

---

## 📊 Dashboard Highlights

The dashboard includes:

### 🔹 Top Run Contributors
- Identifies leading batsmen based on total runs
- Highlights dominance of top-order players

### 🔹 Player Performance Comparison (Runs vs Impact)
- Visual comparison of key players using bubble chart
- Helps evaluate performance beyond raw totals

### 🔹 Extras Conceded by Bowlers
- Analyzes distribution of extras among bowlers
- Provides context on bowling under pressure situations

---

## 🧠 Key Objectives

- Analyze player performance trends across seasons  
- Identify undervalued players for auction strategies  
- Understand contextual performance (e.g., death overs, powerplay)  
- Translate data into actionable insights for decision-making
